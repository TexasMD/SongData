# Security Notice: Do Not Share GitHub Tokens

Please **do not** provide me with a GitHub Personal Access Token (PAT) or a GitHub App Token in our chat.

Sharing tokens in chat logs or directly with AI assistants is a significant security risk, as those logs may be stored or reviewed, potentially exposing your credentials.

Furthermore, even if you provided the token, my execution environment (this sandbox) does not have the `gh` (GitHub CLI) tool installed, so I would not be able to use it effectively anyway.

## How to Proceed

The safest and most effective way for us to work together is for you to simply **copy and paste the text of the assignment or issue** directly into our chat. I can then read it and execute the necessary code changes within this secure sandbox.
